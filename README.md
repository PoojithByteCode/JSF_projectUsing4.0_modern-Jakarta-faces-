UserPortal JSF 4.0 - Employee Management System
📌 Project Overview

UserPortal_JSF is an Employee Management System developed using Jakarta Faces (JSF) 4.0, Java 17, CDI (Weld), JDBC, MySQL, and Apache Tomcat 10.1. The application demonstrates a complete employee registration and login workflow while following a layered architecture (Bean → DAO → Database).

The project is being developed module by module to understand enterprise Java application development from scratch.

✨ Features
Employee Registration
Employee Login Authentication
JDBC Database Connectivity
Session Management
Logout Functionality
Dashboard Access Protection
Required Field Validation
Duplicate Employee ID Validation
Duplicate Email Validation
Successful Registration Message
Invalid Login Message
Windows XP Style User Interface
